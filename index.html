// what it does

Four things, done well.

We're not building another IDE. Nightowl is the layer between you and the mess of tools you already use.

01 / watch

Smart file watching

Detects what changed, decides what to run. No config file bloat. If your project has a package.json, Nightowl already knows what to do.

nightowl watch --smart
02 / sync

Cross-device sessions

Start on your laptop at the office, continue on your desktop at home. Your terminal state travels with you.

nightowl sync --to=home
03 / recall

Command history that thinks

Search across every command you've ever run, filtered by project, by day, by outcome. "What did I type last Tuesday?" is a real question now.

nightowl recall "docker"
04 / share

Shareable sessions

Debug with a teammate without screen-sharing. Send them a terminal snapshot they can inspect, replay, and comment on.

nightowl share --last
